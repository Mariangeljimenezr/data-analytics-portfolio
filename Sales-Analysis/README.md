# Sales Performance Analysis

## Project Overview
This project analyzes sales performance across different regions, product categories, and customer segments in order to identify key drivers of revenue and profitability.

The objective is to support data-driven decision making by identifying patterns in sales performance and highlighting opportunities to optimize business strategy.

## Business Questions
- Which product categories generate the highest revenue?
- Which regions are the most profitable?
- Which customer segments contribute the most to total sales?
- Are there patterns in discounts that affect profitability?

## Dataset
The dataset used for this analysis is the **Superstore dataset**, commonly used for business analytics and data visualization projects.

## Tools Used
- SQL
- Excel
- Power BI
- Data visualization techniques

## Expected Deliverables
- Data cleaning and transformation
- Exploratory data analysis
- SQL queries for key metrics
- A Power BI dashboard showing sales performance
- Business insights and recommendations
